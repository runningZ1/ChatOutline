# 图标文件说明

此目录需要放置以下尺寸的图标：

- `icon-16.png` - 16x16 像素
- `icon-48.png` - 48x48 像素
- `icon-128.png` - 128x128 像素

## 图标设计建议

1. **主题**：使用列表/大纲相关的图标
2. **颜色**：建议使用紫色渐变（与悬浮按钮风格一致）
3. **风格**：简洁现代，确保在小尺寸下清晰可辨

## 临时解决方案

在正式图标制作完成前，你可以：
1. 使用在线图标生成工具创建临时图标
2. 或者暂时注释掉 manifest.json 中的图标配置

## 推荐工具

- [Figma](https://www.figma.com/)
- [Canva](https://www.canva.com/)
- [IconKitchen](https://icon.kitchen/)
