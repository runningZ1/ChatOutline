# 🚀 如何加载和测试插件

## 构建已完成！

插件已成功构建到 `dist` 目录。

## 加载插件到浏览器

### Chrome / Edge 浏览器

#### 步骤 1: 打开扩展管理页面

在浏览器地址栏输入以下地址之一：

- **Chrome**: `chrome://extensions/`
- **Edge**: `edge://extensions/`

或者通过菜单打开：
- Chrome: 右上角 ⋮ → 扩展程序 → 管理扩展程序
- Edge: 右上角 ⋯ → 扩展 → 管理扩展

#### 步骤 2: 开启开发者模式

在扩展管理页面的右上角，找到并**开启「开发者模式」**开关。

#### 步骤 3: 加载插件

点击左上角的「**加载已解压的扩展程序**」按钮。

#### 步骤 4: 选择 dist 目录

在弹出的文件选择对话框中，导航到：

```
D:\Programs\ChatOutline\dist
```

选择这个 `dist` 文件夹，点击「选择文件夹」。

#### 步骤 5: 完成！

你应该看到插件已经出现在扩展列表中：
- 名称：**ChatOutline - AI对话大纲导航**
- 状态：已启用

## 🧪 测试插件

### 1. 访问 ChatGPT

打开 https://chatgpt.com 或 https://chat.openai.com

### 2. 开始对话

发送几条消息，例如：
- "你好，请介绍一下自己"
- "什么是人工智能？"
- "如何学习编程？"

### 3. 查看大纲

在页面右下角，你会看到一个**紫色渐变的圆形悬浮按钮**。

点击这个按钮，大纲面板会从右侧滑出，显示你所有的提问。

### 4. 测试跳转

点击大纲中的任意一项，页面会自动滚动到对应的对话位置，并有高亮动画效果。

## 🔍 查看调试信息

如果遇到问题，按 **F12** 打开开发者工具，切换到 **Console** 标签。

所有插件的日志都以 `[ChatOutline]` 开头，例如：
- `[ChatOutline] 内容脚本已加载`
- `[OutlineManager] 检测到平台: ChatGPT`
- `[ChatGPT Parser] 找到用户消息数: 3`

## ⚠️ 常见问题

### 问题：看不到悬浮按钮

**解决方法**：
1. 刷新页面（Ctrl+R 或 Cmd+R）
2. 确认插件已启用
3. 查看控制台是否有错误信息

### 问题：大纲显示"暂无对话消息"

**解决方法**：
1. 确保已经发送了至少一条消息
2. 等待 1-2 秒让页面完全加载
3. 刷新页面重试

### 问题：修改代码后不生效

**解决方法**：
1. 重新运行 `npm run build`
2. 回到扩展管理页面
3. 点击插件卡片上的「重新加载」按钮 🔄
4. 刷新测试页面
